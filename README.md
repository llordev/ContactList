# Aplicación de escritorio: ContactList.

# Funciones:

- Puedes añadir contactos.
- Puedes buscar contactos.
- Puedes eliminar contactos.

# Desarrollo:

- Esta aplicación fue desarrollada como un proyecto con un objetivo educativo, puede contener errores y contener márgenes de mejora, tenga en cuenta que el desarrollador no tiene conocimientos avanzados ni mucha experiencia.

# Características técnicas:

- Validación de los datos introducidos por el usuario de manera muy simplificada.
- No permite hacer llamadas.

# Tecnologías:

- Lógica desarrollada con Python.
- Interfaces gráficas desarrolladas con PySide6.
- Almacenamiento y gestión de datos desarrollada con SQLite3.

# Para la descarga y ejecución del programa siga los siguientes pasos:

1. Clonar repositorio.
2. Crear espacio virtual.
3. Descargar dependencias necesarias.
4. Ejecutar archivo: main.py.
